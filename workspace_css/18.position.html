1.position:컨텐츠의 위치를 어디에?

  • -statick:(default)
  • -fixed:화면 전체를 기준으로 고정, 스크롤되어도 움직이지 않음
  • -absolute:고정(화면좌상단 기준), 스크롤되면 움직임
  • -relative:static 기준에서 상대적으로 치조정
  • *fixed,absolute,-relatve
    ->left,right,top,bottom
  • *포지션을 변경하면 대부분의 태그는 블록요소로 변경
  • *asolute속성값은 통상적으로 relative속성값과 께 씀

2.overflow:컨텐츠가 영역의 범위를 초과할때 어떻게?

  • overflow:visivle:(default)
  • overflow:hidden:넘치는 부분은 보이지 않게 하겟다
  • overflow:scroll: 스크롤생성
  • overflow:auto: 범위를 벗어날때만 스크롤로 생성

3.요소 (태그)보이기 숨기기

  • display:none; -> 요소가 사라지고 공간도사라짐
  • visibility:hidden; -> 공간은 유지하고 요소만 사라짐
  • opacity:0~1; -> 투명도조절 공간유지
